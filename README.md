# Sistem Perpustakaan

## Deskripsi
Program ini dibuat untuk mengimplementasikan struktur data Dictionary pada Python.

## Fitur
- Menampilkan daftar buku
- Menambahkan buku
- Meminjam buku
- Mengembalikan buku

## Konsep yang Digunakan
- Dictionary
- Perulangan (while)
- Percabangan (if-else)

## Cara Menjalankan
1. Jalankan file main.py
2. Pilih menu sesuai kebutuhan
